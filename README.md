# TinyFS
Simulating a file system.

## Requirement
- python3

## Run
``` bash
python3 main.py
```

## Commands
- mkdir
- ls
- cd
- read (file)
- create (file)
- overwrite (file)
- append (file)
- rm (file in cwd)
- rmdir (empty directory)
- pwd
- save (as a .pkl file)
- load (saved name)
- exit
- clear
